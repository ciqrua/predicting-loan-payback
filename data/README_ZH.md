# 数据集说明

本项目使用的数据来自 Kaggle Playground 系列中的贷款数据集，属于结构化表格数据。

## 数据来源

- Kaggle Playground Series（Season 5, Episode 11）
- 任务目标：预测一笔贷款是否能够被成功还清（`loan_paid_back`）

出于数据许可及仓库体积的考虑，原始数据文件 **未直接包含** 在本 GitHub 仓库中。

## 目录中应包含的文件

在从 Kaggle 下载数据集后，`data/` 目录下应包含以下文件：

- `train.csv`：训练数据，包含特征与目标变量
- `test.csv`：测试数据，仅包含特征，不包含目标变量

## 说明

- 数据集同时包含数值型特征与类别型特征
- 缺失值的处理方式已在主 Notebook 中实现
- 不同运行环境下可能需要根据实际情况调整文件路径
